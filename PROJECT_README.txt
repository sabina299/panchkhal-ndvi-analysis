MULTI-TEMPORAL NDVI ANALYSIS
Panchkhal Region, Kavre District, Nepal
March 2026

PROJECT OVERVIEW
This project analyzes agricultural vegetation health using satellite remote 
sensing to monitor crop development across the 2025 growing season.

STUDY AREA
Panchkhal and surrounding agricultural areas, Kavre District, Nepal
Approximately 2,400 hectares analyzed

DATA SOURCES
- Sentinel-2 Level 2A satellite imagery (European Space Agency)
- Acquisition dates: May 16, September 25, October 13, 2025
- Spatial resolution: 10 meters

METHODS
1. Downloaded Sentinel-2 imagery from Copernicus Data Hub
2. Processed Red (Band 4) and Near-Infrared (Band 8) spectral bands
3. Calculated NDVI using formula: (NIR - Red) / (NIR + Red)
4. Created GeoTIFF files for spatial analysis
5. Developed professional cartographic visualizations in QGIS

TOOLS USED
- Python (rasterio, numpy, pandas) - Data processing and NDVI calculation
- QGIS - Cartographic visualization and spatial analysis
- Sentinel Hub - Data access and download

KEY FINDINGS
- Mean NDVI increased from 0.326 (May) to 0.478 (October)
- Progression shows healthy crop development through growing season
- September showed peak vegetation health (Mean NDVI: 0.438)
- Spatial analysis identified areas with variable vegetation health

OUTPUTS
- 3 professional NDVI maps (May, September, October 2025)
- Summary statistics (CSV format)
- Time-series graph showing seasonal vegetation progression
- Automated Python processing code
- GeoTIFF files for further analysis

APPLICATIONS
- Precision agriculture planning
- Crop health monitoring
- Variable rate management zone identification
- Seasonal vegetation analysis
